## Grading System

- **A+** - Elite (70%+ confidence)
- **A** - Excellent (60%+ confidence)  
- **B+** - Strong (50%+ confidence)
- **B** - Good (40%+ confidence)
- **C+** - Fair (30%+ confidence)
- **C** - Decent (20%+ confidence)
- **D** - Risky (15%+ confidence)
- **F** - Avoid (<15% confidence)
